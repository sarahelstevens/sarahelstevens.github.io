Punters Web Profiles

In line with the mobile web and app upgrades to all profiles pages, the user profile pages required a desktop redesign to enhance the social betting strategy.

The most searched category

Users search other people's profile pages to monitor the betting and tipping performance of their peers. As part of the desktop upgrade Punters introduced private messaging functionality.

Immediately after launching private messaging, users began communicating more frequently and in new and different ways. The user uptake was significant as they chose to exchange tips and advice in confidence.
